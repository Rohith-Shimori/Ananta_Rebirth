# 🚀 START HERE - Ananta Rebirth Quick Guide

## ⚡ 5-Minute Quick Start

### 1️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 2️⃣ Start Ollama (in separate terminal)
```bash
ollama serve
```

### 3️⃣ Run Quick Test
```bash
python quick_test.py
```

### 4️⃣ Start Using Ananta
```bash
# Terminal interface
python main.py

# OR GUI interface
python gui_launcher.py
```

```bash

```

### GPU Issues
Check: `nvidia-smi`
