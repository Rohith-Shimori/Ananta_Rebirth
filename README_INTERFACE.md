## 🛠️ **Installation & Launch**

### **1. Install Dependencies**
```bash
cd C:/Ananta_Rebirth
pip install -r requirements.txt
```

### **2. Launch Interface**
```bash

python gui_launcher.py

# Or Direct Mode
python ui/main_window.py
```

### **3. Terminal Mode (Backup)**
```bash
python main.py
```
